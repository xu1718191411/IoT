# iot_project_ios
This is the app for connecting iot server

***requirements***
- Flutter 2.2.2 
- Dart 2.13.3
- libprotoc 3.12.3


***generate grpc***

only necessary if you modify proto file

```
cd lib/grpc
make grpc
```


***Reference***:
- [grpc dart](https://grpc.io/docs/languages/dart/quickstart/)
