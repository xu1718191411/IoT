#IoT

***requirements***
- Python 3.6.2
- Flask 1.1.4

***run server***

```
python server.py
```

```
cd grpc_server
python server_streaming.py
```

***generate grpc***

only necessary if you modify proto file

```
cd grpc_server
make grpc
```
